# Root Reaper 🌳💀

*Root Reaper* is a Low-Poly, 3D Arcade game where Little Timmy tries to survive for as long as he can in a world infected by hostile roots. 

Using the power of slashing, try to eliminate infected roots as much as possible to get the highest score, use power-ups to help you, and most importantly, stay alive.

This game was made as a project submission for the Game Engines 1 course at De La Salle University's Game Development program (2024).

## 💻 Installation
To play the game on your local machine:

1. Download the latest release from the [Releases page](https://github.com/AEzirrr/Root-Reaper-Repo/releases/tag/v1.0.0) OR download from [itch.io](https://vincent-chang.itch.io/rootreaper).
2. Extract the `.zip` file to your preferred folder.
3. Run the `RootReaper.exe` file.
4. Try to survive!
